# Web Media Scraper: Extract Image and Audio URLs

This small project demonstrates how to scrape image URLs and audio URLs from two public websites using Python, Requests, and BeautifulSoup.

## Features
- Extract image URLs from: https://books.toscrape.com/
- Extract audio URLs from: https://www.soundhelix.com/audio-examples
- Clean and store results in Python lists
- Save final URLs into CSV files (optional)

## Tools Used
- Python
- BeautifulSoup
- Requests
- Google Colab (optional)

## How to Run
1. Open the Jupyter Notebook (`webscraping.ipynb`) in Google Colab or Jupyter.
2. Run all the cells.
3. The notebook will:
   - Scrape image links from Books to Scrape
   - Scrape audio links from SoundHelix
   - Print sample results

## Outputs
- A list of image URLs
- A list of audio URLs
- You can also export them into CSV files

## Purpose
This is a beginner-friendly web scraping project suitable for GitHub portfolios.
